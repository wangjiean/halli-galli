## 🎲 游戏规则

### 经典模式 (60 张牌)
- 水果：🍌 香蕉、🍓 草莓、🍋 柠檬、🟣 李子
- 按铃条件：**5 张同种水果**

### 极限模式 (72 张牌)
- 水果牌 64 张 + 动物牌 8 张 (🐵 猴子×3, 🐘 大象×3, 🐷 猪×2)
- 按铃条件（5 种）：
  1. 5 张同种水果
  2. 5 张动物牌
  3. 4 种不同水果/动物
  4. 2 对相同的牌
  5. 连续数字的水果牌

### 游戏流程
1. 创建/加入房间 → 准备 → 开始游戏
2. 轮流出牌到中央牌堆
3. 发现满足条件时按铃
4. 正确：赢得中央所有牌；错误：失去 1 张牌给对手
5. 赢得所有牌的玩家获胜

## 🔌 接口设计

### REST API
- `POST /api/auth/register` - 注册
- `POST /api/auth/login` - 登录
- `POST /api/scores` - 提交分数 (需认证)
- `GET /api/leaderboard` - 查询排行榜

### Socket.IO 事件
**房间事件**: `room:create`, `room:join`, `room:leave`, `room:ready`, `room:start`, `rooms:list`

**游戏事件**: `game:play-card`, `game:bell`, `game:bell-result`, `game:start`, `game:over`
