## 🚀 快速开始

### 第一步：访问游戏
打开浏览器，访问：**http://localhost:5173**

### 第二步：注册/登录
首次游玩需要注册账号，已有账号直接登录。

### 第三步：选择模式
选择「经典模式」或「极限模式」进入大厅。

### 第四步：开始游戏
创建房间或加入现有房间，准备后开始游戏！

## 📝 注册账号

### 注册步骤

1. **进入注册页面**
   - 打开游戏首页后，点击「注册」按钮
   - 或直接访问：http://localhost:5173/register

2. **填写信息**
   ```
   用户名：3-20 个字符（字母、数字、下划线）
   密码：6-20 个字符
   确认密码：再次输入相同密码
   ```

3. **完成注册**
   - 点击「注册」按钮
   - 注册成功后自动跳转到登录页

### 用户名规则
- ✅ 允许：`player123`, `HalliGalli`, `user_001`
- ❌ 不允许：`ab` (太短), `玩家 1` (中文), `verylongusername123456` (超长)

### 密码规则
- ✅ 允许：`123456`, `password`, `Abc@123`
- ❌ 不允许：`12345` (太短)

### 注意事项
- 用户名不能重复
- 密码需要两次输入一致
- 注册成功后请牢记账号信息

## 🔐 登录游戏

### 登录步骤

1. **进入登录页**
   - 访问：http://localhost:5173/login

2. **输入账号信息**
   ```
   用户名：你注册的用户名
   密码：你设置的密码
   ```

3. **开始游戏**
   - 点击「登录」按钮
   - 登录成功后自动跳转到主页

### 登录状态
- 登录状态保持 7 天
- 7 天后需要重新登录
- 点击「退出」按钮可主动退出登录

### 忘记密码
目前不支持密码找回功能，请妥善保管账号信息。

## 🎯 选择游戏模式

### 经典模式 (Classic) 🍌
**适合新手和休闲玩家**

- **牌数**: 60 张
- **按铃条件**: 仅 1 种 - 5 张同种水果
- **难度**: ⭐⭐
- **推荐**: 新手入门、轻松娱乐

### 极限模式 (Extreme) 🐵
**适合进阶玩家**

- **牌数**: 72 张（64 水果 + 8 动物）
- **按铃条件**: 5 种（见下方详解）
- **难度**: ⭐⭐⭐⭐⭐
- **推荐**: 挑战自我、竞技对战

### 如何选择
1. 在主页点击对应的模式卡片
2. 经典模式点击「🍌 经典模式」
3. 极限模式点击「🐵 极限模式」

## 🏠 创建/加入房间

### 创建房间

1. **进入大厅**
   - 点击模式卡片后进入游戏大厅

2. **设置房间**
   ```
   房间名称：自定义名称（必填）
   游戏模式：经典/极限（进入大厅前已选择）
   最大玩家数：2-6 人（滑块调整）
   ```

3. **创建成功**
   - 点击「创建房间」按钮
   - 自动进入房间等待界面

### 加入房间

1. **浏览可用房间**
   - 大厅中显示所有可用房间
   - 查看房间名称、模式、人数

2. **加入房间**
   - 找到想加入的房间
   - 点击「加入」按钮

3. **进入房间**
   - 加入成功后进入房间等待界面

### 房间状态说明
- **等待中**: 房间正在等人加入
- **已满**: 房间人数已满，无法加入
- **游戏中**: 游戏已开始，无法加入

## 🎮 游戏操作

### 准备阶段

1. **等待对手**
   - 房间显示所有玩家列表
   - 房主名字旁有「房主」标签

2. **准备游戏**
   - 点击「准备」按钮
   - 按钮变绿色表示已准备
   - 再次点击可取消准备

3. **开始游戏**
   - 所有玩家准备后，房主点击「开始游戏」
   - 3 秒倒计时后游戏开始

### 游戏进行中

#### 出牌
**当轮到你时**（你的名字旁有 ▶ 标志）：

1. **点击手牌**
   - 底部显示你的手牌
   - 点击任意一张牌出牌

2. **牌到中央**
   - 出的牌会放到中央牌堆
   - 回合自动交给下一位玩家

#### 按铃
**任何时候**（不需要轮到你）：

1. **观察中央牌堆**
   - 注意牌堆最上方的几张牌
   - 判断是否满足按铃条件

2. **点击按铃按钮**
   - 点击红色的「🔔 按铃！」按钮
   - 按铃后有 0.2 秒冷却时间

3. **判定结果**
   - ✅ 正确：赢得中央所有牌
   - ❌ 错误：被罚一张牌给对手

### 游戏结束

1. **胜利条件**
   - 赢得所有牌的玩家获胜

2. **结算界面**
   - 显示「🎉 胜利！」或「😢 失败」
   - 点击「返回大厅」回到大厅

## 🔔 按铃条件详解

### 经典模式（1 种条件）

#### 🍌 5 张同种水果
中央牌堆最上方的 5 张牌都是同一种水果

**示例**：
```
🍌 🍌 🍌 🍌 🍌  ← 5 个香蕉，可以按铃！
🍋 🍋 🍋 🍋 🍋  ← 5 个柠檬，可以按铃！
🍌 🍋 🍌 🍋 🍌  ← 混合水果，不能按铃
```

---

### 极限模式（5 种条件）

#### 1️⃣ 5 张同种水果
同上，5 张相同水果

**示例**：
```
🍓 🍓 🍓 🍓 🍓  ← 5 个草莓，可以按铃！
```

#### 2️⃣ 5 张动物牌 🐵
中央牌堆最上方的 5 张牌都是动物

**动物种类**：🐵 猴子、🐘 大象、🐷 猪

**示例**：
```
🐵 🐘 🐷 🐵 🐘  ← 5 张动物牌，可以按铃！
🐵 🍌 🐘 🐷 🐵  ← 混入水果，不能按铃
```

#### 3️⃣ 4 种不同 🎨
中央牌堆最上方的 4 张牌包含 4 种不同的水果/动物

**示例**：
```
🍌 🍋 🍓 🐵  ← 香蕉、柠檬、草莓、猴子，4 种不同，可以按铃！
🍌 🍋 🍌 🍓  ← 只有 3 种（香蕉重复），不能按铃
```

#### 4️⃣ 2 对牌 👯
中央牌堆最上方的 4 张牌包含 2 对相同的牌

**示例**：
```
🍌 🍌 🍋 🍋  ← 2 对（香蕉对 + 柠檬对），可以按铃！
🍌 🍌 🍌 🍋  ← 只有 1 对（3 个香蕉），不能按铃
🍌 🍋 🍓 🐵  ← 没有对子，不能按铃
```

#### 5️⃣ 连续数字 🔢
中央牌堆最上方的 3 张**水果牌**的数量连续递增

**牌面数字**：每张水果牌上有数字（1-5）

**示例**：
```
🍌₂ 🍋₃ 🍓₄  ← 2-3-4 连续，可以按铃！
🍌₁ 🍋₂ 🍓₃  ← 1-2-3 连续，可以按铃！
🍌₂ 🍋₄ 🍓₅  ← 2-4-5 不连续，不能按铃
🍌₃ 🍋₃ 🍓₄  ← 3-3-4 有重复，不能按铃
```

**注意**：动物牌没有数字，不参与连续判断

## 🏆 胜负规则

### 胜利条件
- **赢得所有牌**：当你拥有全部 60 张（经典）或 72 张（极限）牌时获胜

### 游戏流程
1. 每位玩家初始获得 30 张牌（经典）或 36 张牌（极限）
2. 轮流将手牌出到中央
3. 满足条件时按铃赢得中央所有牌
4. 先拥有所有牌的玩家获胜

### 积分规则（排行榜）
- **胜场**: 获胜次数 +1
- **游戏场次**: 每局游戏 +1
- **胜率**: 胜场 ÷ 游戏场次 × 100%

### 排行榜
- 主页点击「排行榜」查看
- 按胜场数降序排列
- 显示前 10 名玩家

## ❓ 常见问题

### Q1: 按铃后为什么没有反应？
**A**: 可能原因：
- 按铃过于频繁（有 0.2 秒冷却）
- 按铃时中央牌堆为空
- 网络延迟

### Q2: 为什么我被罚牌了？
**A**: 错误按铃会被罚牌：
- 不满足条件时按铃
- 按铃条件判断错误

### Q3: 回合制是怎样的？
**A**: 
- 顺时针轮流进行
- 你的名字旁有 ▶ 标志时才能出牌
- 出牌后自动交给下一位

### Q4: 可以中途退出吗？
**A**: 
- 可以点击「离开房间」退出
- 游戏进行中退出视为放弃
- 房间会在 5 分钟无人后自动清理

### Q5: 断线了怎么办？
**A**: 
- 重新登录即可
- 房间会保留一段时间
- 尽快重新加入

### Q6: 如何知道该不该按铃？
**A**: 
- 时刻观察中央牌堆
- 心中默念当前牌的水果/动物种类
- 多练习提高反应速度

### Q7: 经典模式和极限模式哪个好玩？
**A**: 
- 新手建议从经典模式开始
- 熟练后挑战极限模式
- 极限模式更有挑战性

### Q8: 为什么我创建不了房间？
**A**: 
- 检查是否已登录
- 房间名称不能为空
- 网络是否正常

### Q9: 加入房间失败？
**A**: 
- 房间可能已满
- 房间可能已开始游戏
- 刷新大厅重新尝试

### Q10: 游戏卡顿怎么办？
**A**: 
- 检查网络连接
- 刷新页面
- 重启浏览器

## 🎯 游戏技巧

### 新手技巧
1. **集中注意力**: 时刻关注中央牌堆
2. **不要急**: 确认满足条件再按铃
3. **记牌**: 记住已经出了哪些牌

### 进阶技巧
1. **预判**: 根据手牌预测可能出现的组合
2. **心理战**: 假装要按铃迷惑对手
3. **快速反应**: 看到条件立即按铃

### 极限模式技巧
1. **优先判断动物**: 5 张动物牌容易识别
2. **注意数字**: 连续数字需要记住牌面
3. **组合思考**: 同时判断多种条件
